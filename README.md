## Exécuter localement

Cloner le projet

```bash
  git clone https://github.com/Totore-7/agenda
```

Aller dans le répertoire du projet

```bash
  cd agenda
```

Crée un ficher /js/config.js pour vous connecter

```js
export default {
    USERNAME: '-username-',
    PASSWORD: '-password-',
  };
```

Utiliser l'extention "live server" de vs code pour faire tourner le site
